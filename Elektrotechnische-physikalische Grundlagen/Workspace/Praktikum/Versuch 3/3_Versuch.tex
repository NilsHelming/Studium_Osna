Stromfehlerschaltung mit $R_x = 2,7\unit{\textOmega}$:\\
Es wurde ein Strom von $500\unit{mA}$ une eine Spannung von $1,36\unit{V}$ gemessen. Somit würden wir einen Widerstand $R = \frac{U}{I} = \frac{1,36\unit{V}}{500\unit{mA}} = 2,72\unit{\textOmega}$ erwarten.

Stromfehlerschaltung mit $R_x = 10\unit{k\textOmega}$:\\
Hier haben sich Messungen von $4,75\unit{V}$ und $0,5\unit{mA}$ ergeben. Damit hätten wir einen Widerstand von $R = \frac{U}{I} = \frac{4,75\unit{V}}{0,5\unit{mA}} = 2,72\unit{\textOmega}$ gemessen.
